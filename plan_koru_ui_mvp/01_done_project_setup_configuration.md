# Step 01: Project Setup & Configuration

## Objective
Set up React 18+ TypeScript with Vite, Tailwind CSS, and PWA support for the Koru meditation app.

## Key Tasks

### 1.1 Install Dependencies
**Framework**: `react`, `react-dom`, `react-router-dom`
**Build**: `vite`, `@vitejs/plugin-react`, `typescript`, `@types/react`, `@types/react-dom`
**Styling**: `tailwindcss`, `postcss`, `autoprefixer`, `@tailwindcss/forms`, `tailwindcss-animate`
**PWA**: `vite-plugin-pwa`, `workbox-window`

Install with: `pnpm add [packages]` (use `-D` for dev dependencies)

### 1.2 Configure TypeScript
**File**: `tsconfig.json`

Enable strict mode, target ES2020, JSX as `react-jsx`, path alias `@/*` → `./src/*`, include noUnusedLocals/noUnusedParameters

### 1.3 Configure Vite
**File**: `vite.config.ts`

Add React plugin, PWA plugin with manifest settings, path alias (`@` → `./src`), dev server port 5173 with auto-open. PWA Workbox: Runtime caching for Google Fonts (CacheFirst, 1 year), asset precaching for fonts/icons.

### 1.4 Configure Tailwind
**File**: `tailwind.config.js`

**Content**: `./index.html`, `./src/**/*.{js,ts,jsx,tsx}`

**Theme extensions**: Colors (peach, warm, calm, gentle - see UI_design.md §3 for hex codes), fonts (Lora serif, Inter sans), custom spacing (18, 22), border radius (card, card-lg), shadows (card, card-hover, elevated), session text sizes.

**Plugins**: `@tailwindcss/forms`, `tailwindcss-animate`

**Init**: `pnpm dlx tailwindcss init -p`

### 1.5 Create Base Styles
**File**: `src/styles/globals.css`

Import Tailwind directives, import Lora + Inter from Google Fonts. Base styles: bg-warm-50, text-calm-900, font-sans, dynamic viewport height (100dvh), headings use font-serif. Add focus-visible styles (peach ring). **CRITICAL**: Implement `@media (prefers-reduced-motion: reduce)` to disable animations (see UI_design.md §3). Add safe-area inset utilities.

### 1.6 Create Entry Points
**src/main.tsx**: React root with StrictMode, import App + global styles

**src/App.tsx**: Simple component showing "Koru" in peach color (verification only, will add providers/router later)

**index.html**: Viewport with `viewport-fit=cover`, theme-color meta (#FF9A54), description, title "Koru - Meditation Rituals"

### 1.7 Package Scripts
**package.json**: Add `dev` (Vite dev server), `build` (TypeScript + Vite build), `preview` (production preview), `lint` (TypeScript type check)
