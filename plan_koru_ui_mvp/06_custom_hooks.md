# Step 6: Custom Hooks

## Objective
Create reusable React hooks for common operations.

## Files to Create

```
src/hooks/
├── useLocalStorage.ts
├── useReducedMotion.ts
├── useBackgroundTask.ts
├── useNotification.ts
└── useRitual.ts
```

## Hook Descriptions

### useLocalStorage
Typed localStorage hook with automatic JSON serialization.

### useReducedMotion
Detects user's motion preference from OS settings.

### useBackgroundTask
Wrapper around background-task-service for async operations.

### useNotification
Request permission and show notifications.

### useRitual
CRUD operations for rituals (wraps RitualContext).
