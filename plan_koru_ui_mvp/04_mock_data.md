# Step 4: Mock Data

## Objective
Create realistic mock data for rituals and quick starts.

## Files to Create

- `src/mocks/rituals.ts` - 6-8 complete rituals
- `src/mocks/quickStarts.ts` - 6 quick start rituals

## Mock Rituals Requirements

Each ritual should include:
- Complete metadata (title, intent, duration, tone, pace)
- 3-5 sections with full guidance text
- Realistic durations (5-20 minutes total)
- Various tones: gentle, neutral, coach
- Tags: Focus, Calm, Confidence, Gratitude, Sleep, Breath, Body

### Example Quick Starts

1. **Reset** (3 min) - "For anxious moments" - Breath tag
2. **Focus Primer** (5 min) - "Before deep work" - Focus tag
3. **Wind-Down** (10 min) - "End your day" - Sleep tag
4. **Gratitude** (7 min) - "Shift perspective" - Gratitude tag
5. **Confidence** (8 min) - "Before challenges" - Confidence tag
6. **Silent Timer** (5-20 min) - "Pure silence" - Meditation tag
